Este projeto apresenta uma tradução dos Evangelhos do Aramaico Siríaco para o Português do Brasil, com o objetivo de fornecer uma versão acessível e fiel aos textos originais.

## 🚀 Funcionalidades Principais

- **Tradução Direta:** Textos traduzidos diretamente do Aramaico Siríaco para o Português do Brasil.
- **Manuscritos Antigos:** Baseado em manuscritos como os Evangelhos Curetonianos Siríacos, o Palimpsesto Sinaítico Siríaco Antigo e a Peshitta.
- **Navegação Intuitiva:** Interface simples para navegar entre livros, capítulos e versículos.
- **Notas do Tradutor:** Informações adicionais e notas explicativas para melhor compreensão.
- **Design Responsivo:** Compatível com diversos dispositivos, como desktops, tablets e smartphones.
- **Tooltips:** Informações adicionais sobre palavras específicas ao passar
